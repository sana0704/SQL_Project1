# Hospital Management System - SQL Practice

This project is a simple SQL-based hospital management system for practicing basic database operations like CREATE, INSERT, SELECT, JOIN, UPDATE, and DELETE.

## Tables Included
- Doctor
- Patients
- Appointments

## Sample Features
- Add and manage doctor and patient records
- Link patients with appointments
- Perform SQL queries to retrieve and update information

## Practice Topics
- CRUD operations
- Joins
- Filtering and sorting
- Aggregation
